## Задание №1

Cбор HTML-страниц с сайта dic.academic.ru.

### Требования
- Python
- Библиотека requests

### Установка и запуск

1. Сохраните файл `main.py` в любую папку

2. Откройте терминал/командную строку и перейдите в папку с программой:
```bash
cd путь/к/папке/с/main.py
```

3. Установите необходимую библиотеку:
```bash
pip install requests
```

4. Запустите программу:
```bash
python main.py
```

### Результат работы
После выполнения программы в папке появятся:

Папка `pages/` - содержит 100 HTML-файлов:
```text
page_1.html

page_2.html

...

page_100.html
```

Файл `index.txt` - соответствие номеров файлов и URL:

```text

2       https://dic.academic.ru/dic.nsf/enc_biology/1401/
...
100     https://dic.academic.ru/dic.nsf/enc_biology/1499/
```
